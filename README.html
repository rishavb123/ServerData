Clips

Each clip describes a few features. The name of the file indicates the feature that it talks about. The number in the name tells you what order it shows up in the full video. To view these videos open Video/clips and then choose the clip to watch.

Website

This is the website that is being hosted at data.bhagat.io. Opening the index.html file will not load the website since the site was built with react and the way react gets its resources in index.html does not work when you open the file. If you would like to see the website from the Website folder, serve it through a server using a command like serve.

Install Dependencies

cd React Source
npm install

Start Development Server

cd React Source
npm start

Create Build

cd React Source
npm run build

Server

The server uses python 3.7 to serve the content. It also require a few modules installed in python: (firebase_admin, sklean). The server will also not work without the credentials to connect to the firebase admin sdk. If it is absolutely required to run the server, email me at rishav@bhagat.io
